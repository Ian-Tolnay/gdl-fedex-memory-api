# Privacy Policy

This API is a private project-memory gateway for the GDL/FedEx engineering workflow.

The service receives project notes, session summaries, validation results, and optional file metadata from an authorized Custom GPT Action. It stores structured records in the Airtable base configured by the service owner.

The service does not sell data, does not share data with advertisers, and does not intentionally expose project records to third parties. Access is protected by a bearer API key configured in the Custom GPT Action and by Airtable permissions.

Do not submit credentials, passwords, API keys, payment card details, or other secrets through this API.
